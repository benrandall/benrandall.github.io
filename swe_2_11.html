CS373 Spring 2018: Benjamin Randall

2/4/2018

What did you do this past week?

In this class I began working on the first phase of the internet data base project. With a little difficulty, we managed to get our group of six together and brainstorm an acceptable idea for the project. We ended up deciding to create a database of RuneScape items, skills, and associated YouTube videos. RuneScape is a fantasy multiplayer online role-playing game released originally in January 2001 by Jagex. RuneScape was hugely popular during the formative years in many of my generation's childhood, so we thought there'd be no better way to honor it than base our project around the game and its culture.

What's in your way?

There are many challenges that present themselves at the beginning of any large project like this. I need to first familiarize myself with the technologies we must use, such as AWS, Flask, and Bootstrap. I also have to work with my team to get an estimate of the scale of this project: how long will it take, how many lines of code? So far, we have been good about splitting up work, but we've only just started.

What will you do next week?

Next week our team will need to finish phase 1 of our project by Thursday. This will include setting up the AWS server and beautifying the front-end. Additionally, I will want to start reviewing my notes and copying them over into a new notebook I got.

What's my experience of the class?

So far, I think the collatz conjecture project was more stressful than this project. I get burnt out working alone on projects, so it is helpful and motivating to have a group to work with. I've been enjoying lectures as always, but I'm not recognizing the large amount of material that will be fair game for the first exam.

What's my pick-of-the-week or tip-of-the-week?

My teammates and I relied heavily on Flasky while setting up the initial Flask boilerplate. Flasky is a comprehensive tutorial on how to creatively use Flask to set up a website complete with basics, tips, and examples. The GitHub repo was particularly useful when trying to figure out syntax and conventions. If you're having trouble with Flask documentation, this book serves as a helpful guide.
